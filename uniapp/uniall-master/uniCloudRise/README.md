## 知识点概述

1. 学习最新的uni-id用户体系，掌握uni-id-pages最新的配置
2. 学习DB Schema概述的语法格式，首页验证规则
3. 掌握JQL数据库操作
4. uniCloud-db前端组件的使用
5. 学习咸虾米的圈子项目案例，数量掌握uniCloud的各个知识模块
6. uni-ui组件库配合uView UI组件库，美化项目页面
